# Ograf Builder

Constructeur de modèles graphiques TV pour OGraf, la nouvelle spécification de graphiques HTML soutenue par l'EBU - fonctionnant directement dans votre navigateur !

> Les graphiques créés avec Ograf Builder sont entièrement compatibles avec OGraf, garantissant une utilisation pérenne sur tout appareil.

## Fonctionnalités

- Concevez et animez vos graphiques HTML comme vous le feriez habituellement.
- Générez des graphiques HTML prêts pour la production et compatibles avec OGraf.
- Exécutez ou déployez vos graphiques OGraf sur tout périphérique de rendu prenant en charge la spécification.
